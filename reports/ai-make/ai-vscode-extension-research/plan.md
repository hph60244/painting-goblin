# Research Plan: AI-Powered VS Code Extension Generation

## Main Research Question
尋找讓Code Agent根據需求文件製作vscode外掛的參考範例
(Find reference examples of using Code Agents to build VS Code extensions from requirement documents)

## Subtopics

### Subtopic 1: AI-Powered VS Code Extension Generation Tools
Investigate existing tools and frameworks that use AI/LLMs to generate VS Code extensions automatically from descriptions or specs.
Expected: Tools like Bolt.new, v0.dev, Replit Agent, or specialized VS Code extension generators.

### Subtopic 2: Examples of Building VS Code Extensions with Cursor/Claude/Copilot
Find real-world examples, blog posts, and tutorials where developers used AI code assistants (Claude, Cursor, GitHub Copilot) to build VS Code extensions.
Expected: Step-by-step guides, lessons learned, and best practices.

### Subtopic 3: Workflow Patterns for Code Agent Extension Development
Research workflow patterns for using agentic coding tools to go from requirements/specs to a working VS Code extension.
Expected: Prompt engineering tips, file structure approaches, and iterative development patterns.

## Synthesis Approach
Compare tools, extract actionable workflows, and provide recommendations for using Code Agents to generate VS Code extensions from requirement documents.
