# Painting Goblin - Current Status and Usage Research

## Project Type

"Painting Goblin" refers to two distinct software projects sharing the same name:

1. **The Painting Goblin Game** - A game developed in Processing (Java-based) that implements utility-based AI, inspired by the Sims "painting goblin" meme. It is a technical exercise in autonomous agent decision-making.

2. **painting-goblin Task Processing System** - A Python-based file-system task queue processor using publisher-subscriber pattern and OpenCode AI for automated task execution. It is a task automation framework for AI-assisted development.
