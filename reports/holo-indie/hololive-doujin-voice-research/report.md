# Hololive 同人遊戲粉絲剪輯語音下載點 — 綜合研究報告

**研究日期:** 2026-05-24

---

## 核心結論

**目前不存在官方授權、可直接下載用於同人遊戲的 Hololive 語音素材庫。** 現有方案分為三類：

1. **官方語音包**（需購買、僅限個人使用，不可嵌入遊戲）
2. **粉絲剪輯語音**（來自公開直播，語音按鈕網站可播放，但授權狀態為灰色地帶）
3. **第三方程式的免費 BGM/SE 素材**（合法可用於 Hololive 主題遊戲）

## 三、Cover 社二創規範 & 語音使用限制

### 3.1 官方規範核心

**官方二創規範網址：** https://hololivepro.com/en/terms/

| 項目 | 內容 |
|------|------|
| **免費同人遊戲** | ✅ 符合規範即可，無需申請許可 |
| **付費/商用遊戲** | ❌ 必須透過 **holo Indie** 計劃 |
| **會員限定/付費內容語音** | ❌ **絕對禁止** |
| **演唱會影片語音** | ❌ **絕對禁止** |
| **AI 語音提取** | ❌ **明確禁止**（歌曲指南中載明） |
| **公開直播 VOD 語音** | ⚠️ 灰色地帶，未明確禁止但也未明確允許 |

### 3.2 holo Indie 計劃（唯一合法商用路徑）

- **申請網址：** https://ccmc-corp.com/en/entry/
- 通過審查後，遊戲可在 Steam 上以 **holo Indie** 品牌發行
- **實例：HoRoyal ~Hololive Battle Royal~**（2026 年 5 月發行）
  - 使用超過 **10,000 個**來自公開遊戲直播的語音片段（GAYA 語音）
  - Steam 頁面：https://store.steampowered.com/app/4449390/
  - 這個案例證明：**透過 holo Indie 計劃，使用直播語音片段是可行的**

### 3.3 語音使用建議分級

```
✅ 最安全：申請 holo Indie → 使用公開直播 VOD 語音 → 免費發行
✅ 較安全：免費同人遊戲 → 使用公開直播語音 → 標註來源（但非官方明文允許）
⚠️ 灰色：免費同人遊戲 → 使用 Nyaa 語音包（官方收費語音重新分發）
❌ 禁止：會員語音 / 演唱會語音 / AI 提取語音 / 任何收費遊戲（無 holo Indie）
```

## 四、具體行動建議

### 如果你要做 Hololive 同人遊戲並使用粉絲剪輯語音：

1. **最佳路徑（官方支援）：**
   - 透過 **holo Indie** 提交申請（https://ccmc-corp.com/en/entry/）
   - 參考 HoRoyal 的方式，從**公開直播 VOD**剪輯語音
   - Steam 免費或付費發行皆可

2. **替代路徑（非商用）：**
   - 從 **101soundboards / Voicy / Myinstants** 下載粉絲剪輯 MP3
   - 從 **HLClips.com** 找到原始剪輯影片，自行提取音訊
   - 免費發行（不可收費），僅限同人/興趣層級

3. **BGM/SE 合法素材：**
   - BGM → **DOVA-SYNDROME**（dova-syndrome.com）
   - 音效 → **効果音ラボ**（soundeffect-lab.info）
   - 角色圖像（RPG Maker）→ **BOOTH ときりす**（免費，二創/商用可）

4. **其他參考資源：**
   - Hololive 粉絲遊戲資料庫：https://softdevwu.dev/hologames/
   - MMD 官方 3D 模型：https://www.mmd.hololive.tv
   - Vchavcha MMD 下載指南：https://vchavcha.com/en/free-resources/hololive-mmd-download/

## 五、注意事項

1. **語音授權狀態多為灰色地帶** — Cover 社的規範主要針對「剪輯影片」而非「遊戲內嵌語音」
2. **會員限定內容的語音絕對不可使用**
3. **AI 複製語音已明確禁止**
4. **官方語音包（Nyaa 上的種子）授權為個人使用，不應嵌入遊戲**
5. **建議在遊戲中標註「This is a fan-made game and is not affiliated with Cover Corp.」**
