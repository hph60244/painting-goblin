## 1. Official Hololive Sources

### hololive.booth.pm — Official Hololive Production BOOTH Store (Closed)

- **Language:** Japanese
- **Description:** The official BOOTH storefront for Hololive Production. Sold official voice packs (voice sets, alarm voices, system sounds) for individual talents including Shirakami Fubuki, Houshou Marine, Usada Pekora, etc. Voice products were available as downloadable WAV files in ZIP archives.

- **Products include:** System voice sets for various talents across JP, EN, ID branches.

## 3. Piapro (ピアプロ) — CGM Content Platform

### Overview
- **URL:** https://piapro.jp/
- **Language:** Japanese
- **Description:** Piapro is a CGM (Consumer Generated Media) content platform operated by Crypton Future Media (creators of Hatsune Miku). Users can post and share music, illustrations, text, and 3D models. Character usage guidelines apply.

### Hololive Presence
- **URL:** https://piapro.jp/hololive
- **Account:** hololive production【公式】(Official)
- **Content:** 317 instrumental tracks (インスト音源) from Hololive original songs/albums. Includes works from Blue Journey, Hololive IDOL PROJECT, etc.
- **Terms:** "改変不可" (no modification allowed) is stated on instrumentals. Follow official Hololive guidelines for any usage (https://www.hololive.tv/terms).
- **Note:** Piapro is primarily for fan creation and sharing. Hololive's account uses it as an instrumental streaming/reference platform rather than an asset distribution platform for game makers.

### Related: Piapro Character Usage Guidelines
- Piapro has a "キャラクター利用のガイドライン" (Character Usage Guidelines) explaining which characters can be used in derivative works. Hololive characters are available through Hololive's own secondary creation guidelines, not through Piapro's character licensing system.

## Key Takeaways for Doujin Game Creators

1. **No single source provides downloadable Hololive talent voice clips licensed for game reuse.** Official voice products are for personal use only and cannot be redistributed or embedded in games.

2. **BOOTH.pm is the best marketplace** for finding doujin-licensed assets (both free and paid) that reference or use Hololive themes, particularly for visual/graphic assets like RPG Maker character sprites.

3. **Voice button sites (VBUP, Myinstants, individual talent buttons)** provide web-playable clips but do not offer downloads, and their licensing status for game use is unclear/unlicensed.

4. **Free BGM/SFX sites** (DOVA-SYNDROME, Sound Effect Lab) are excellent sources for non-Hololive-specific audio assets that can be legally used in Hololive-themed fan games.

5. **Piapro** is primarily for instrumental reference, not asset distribution. Hololive's official account does not distribute downloadable voice banks.

6. **Always consult Hololive's secondary creation guidelines** (https://www.hololive.tv/terms) before using any Hololive-related content in a doujin game.

7. **For voice assets specifically:** doujin game creators may need to either (a) create original voice content, (b) commission voice actors, (c) use royalty-free voice packs from BOOTH, or (d) use voice synthesis software (UTAU, VOICEROID, etc.) to generate character voices, rather than directly using Hololive talent voices.
