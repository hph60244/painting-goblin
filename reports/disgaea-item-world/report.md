# 《魔界戰記 (Disgaea)》系列「道具界 (Item World)」規則與玩法分析報告

---

## 一、什麼是道具界 (Item World)

道具界是《魔界戰記》系列的核心系統之一。遊戲中每一個道具（武器、防具、飾品、消耗品等）內部都存在一個被稱為「道具界」的獨立空間，玩家可以進入其中，通過攻略隨機生成的迷宮來提升道具的能力。此系統自2003年初代作品《魔界戰記》起即存在，貫穿整個系列。

---

## 二、基本規則

### 進入方式
- 從據點的「道具界」入口選擇任意道具進入
- 所有作品都會在第三章提供教學關卡，引導玩家進入道具界

### 樓層推進
- 每一層都是隨機生成的地圖（網格制戰棋地圖）
- 通關方式有兩種：
  1. 擊敗該層所有敵人
  2. 直接前往紅色傳送門進入下一層（可跳過戰鬥）
- **每通關一層，道具等級 +1**

### 樓層數量（依稀有度）
| 稀有度 | 樓層數 |
|--------|--------|
| 普通 (Normal) | 30層 |
| 稀有 (Rare) | 60層 |
| 傳說 (Legendary) | 100層 |

- 從《魔界戰記6》開始，道具界改為**無限樓層制**，不再受限於稀有度
- 道具等級上限：1~5代為300級（D2的修羅模式可達999級），6代以後進一步提升

### 離開方式
- **Mr. Gency's Exit（暫離出口）**：關鍵道具，可在任意時間離開道具界並保留進度
- 若無此道具，只能到達里程碑（每10層）或全滅才能離開

## 三、核心玩法機制

### 1. Geo地板與Geo符號（ゲージパネル・ゲージシンボル）
- 地面存在彩色格子（紅、綠、藍、黃、紫、青）
- Geo符號放在彩色格子上時，會賦予該顏色上所有單位**加成或減益效果**
- 破壞Geo符號會引發連鎖反應：所有受影響的格子轉換顏色，並對站在其上的單位造成傷害
- Geo符號可以和角色一樣被舉起、投擲

### 2. 住人 / Innocents（イノセント）
- 住在道具內的小生物，賦予道具額外能力加成
- 主要住人類型：
  | 住人 | 效果 |
  |------|------|
  | Gladiator（劍士） | 提升ATK |
  | Sentry（哨兵） | 提升DEF |
  | Tutor（導師） | 提升INT |
  | Physician（醫師） | 提升RES |
  | Marksman（射手） | 提升HIT |
  | Coach（教練） | 提升SPD |
  | Dietician（營養師） | 提升HP |
  | Master（大師） | 提升SP |
  | Statistician（統計學家） | 提升獲取經驗值 |
- 住人分為「未馴服」（紅標）和「已馴服」（黃標），只有已馴服的住人可以自由轉移到其他道具
- **特殊住人**：
  - Teacher（教師）：可以提升其他住人的等級
  - Mediator（調解者）：可以將兩個住人合併為雙屬性住人
  - Enforcer（執行者）：可將已馴服的住人轉移到其他道具（消耗自身等級）
  - Collector（收藏家）：提升道具的稀有度數值

### 3. Bonus Gauge（獎勵計量表）
- 通過連鎖攻擊、特殊技能等累積點數
- 過關時根據點數給予額外獎勵道具

### 4. 海盜襲擊（Pirates）
- 回合結束時有一定機率觸發海盜襲擊
- 可能在1~2回合內出現第一波，下一回合有機率出現第二波
- 擊敗海盜可解鎖「創建海盜團」議案
- 擁有海盜團後可進行「Reverse Pirating（逆海盜）」——回到已完成的樓層以獲得高等級黃/紅色等級球

### 5. Boss戰（道具將軍 / 道具王 / 道具神）
- 每10層會出現Boss
- 擊敗Boss會給予道具**數值倍率加成**：
  | 樓層 | Boss | 倍率 |
  |------|------|------|
  | 10F | 道具將軍 | ×1.1 |
  | 20F | 道具將軍 | ×1.1 |
  | 30F | 道具王 | ×1.2 |
  | 40F | 道具將軍 | ×1.1 |
  | 50F | 道具將軍 | ×1.1 |
  | 60F | 道具王 | ×1.2 |
  | 70F | 道具將軍 | ×1.1 |
  | 80F | 道具將軍 | ×1.1 |
  | 90F | 道具王 | ×1.2 |
  | 100F | 道具神 | ×1.3 |
- **雙殺技巧（Double Kill）**：擊敗Boss → 使用Mr. Gency離開 → 重新進入再次擊殺，可獲得兩倍加成
- Boss身上必定持有強力裝備，可用盜賊（Thief）偷取

### 6. 神祕門（Mystery Gate）
- 綠色「?」標記的門，進入後通往隨機房間
- 房間類型：
  - **伏擊房（Ambush Room）**：7隻巨大怪物，2回合內擊敗+5等級，否則+2等級
  - **算命房（Fortune Teller Room）**：隨機獲得大吉(+5級)、吉(+3級)、凶(-3級)、大凶(-5級)，凶以上的結果可激怒算命先生戰鬥取回等級
  - 商人房間：購買稀有道具
- 神秘門會重置當前樓層的配置

### 7. 等級球（Level Sphere）
- 出現於道具界中的特殊球體，被角色舉起並通關後可獲得額外道具等級
  - 綠色：+5級
  - 黃色：+10級
  - 紅色：+15級
- 可自然生成、經由「道具成長路線」生成、或透過Event Area生成

### 8. Event Area（事件區域）
- 紫色柱子標記的區域，需要指定數量的單位站入
- 達成條件後可觸發：寶箱出現、神秘門生成、或等級球生成

### 9. 道具界議會 / Innocent Town（4代以後）
- 每10層有機會進入「Innocent Town」
- 可通過蘑菇NPC提交議案，例如「ATK成長提升」等
- 通過的議案數量與擊敗Boss數量共同決定最終的數值倍率
- 最多可通過6次議案

### 10. 路線分岐（4代）
- 每5層可選擇：
  - **道具成長路線**（劍圖標）：獲得額外等級加成，初期+1級，多次進入後+3級
  - **住人強化路線**（笑臉圖標）：提升住人馴服效果，有機會出現新住人

## 四、系列的演變

### 魔界戰記1（2003）
- 道具界系統的原型首次登場
- 基本的樓層推進 + Geo面板 + 住人系統
- Bonus Gauge獎勵機制

### 魔界戰記2（2006）
- 引入前科（Felony）系統，影響道具界
- 首次引入「修羅之國（Land of Carnage）」高難度版道具界
- 海盜系統首次登場

### 魔界戰記3（2008）
- 新增「職業界（Class World）」——類似道具界但用於強化職業
- 引入Evil Symbol（邪惡符號）系統，如Nether Express提供傳送點

### 魔界戰記4（2011）
- 路線分岐（道具成長 vs 住人強化）
- 道具界議會 / Innocent Town
- 等級球系統
- Event Area系統
- 逆海盜（Reverse Pirating）系統
- 海盜團創建
- 道具等級上限：300

### 魔界戰記 D2（2013）
- 普通/稀有道具可升級為傳說（完成所有樓層後）
- 道具稀有度可改變
- 修羅模式道具等級上限：999

### 魔界戰記5（2015）
- 作弊商店（Cheat Shop）可調整道具界地圖大小（小地圖加速）
- Nether Express邪惡符號提供車站傳送
- 新增道具界研究（自動派遣部隊攻略道具界）

### 魔界戰記6（2021）
- 轉為3D畫面
- 無限樓層制（不再受稀有度限制）
- 道具界研究系統強化（被動升級裝備）
- 支援自動戰鬥（Auto Mode）
- 角色等級上限提升至99,999,999
- 住人/能力系統簡化

### 魔界戰記7（2023）
- 新增**道具轉生（Item Reincarnation）**機制——道具可轉生獲得更多力量
- 超巨大化（Jumbification）影響道具界
- 進一步的品質改善（QoL）

## 五、策略要點

1. **永遠攜帶 Mr. Gency's Exit**：這是道具界的第一黃金法則
2. **盜賊是關鍵角色**：偷取Boss身上的強力裝備
3. **速度跑法**：使用Masked Hero（假面英雄）搭配拳套技能擠開敵人，最高移動力可達31格
4. **雙殺Boss**：最大化數值倍率加成
5. **住人管理**：在開始練道具前先裝滿住人（即使是低等級），Boss戰時每個住人都會提供額外加成
6. **先練泛用裝備**：優先強化Imperial Seal等泛用飾品，而非武器（武器容易被替換）
7. **攜帶補師**：道具界中每層通關不會恢復HP，需要補師維持續戰力
8. **善用逆海盜**：獲取高等級球的關鍵
9. **後期再深入**：前期裝備汰換快，不建議花太多時間在道具界

---

## 六、總結

道具界是《魔界戰記》系列最具代表性的系統之一，提供幾乎無限的成長空間。其核心循環為：選擇道具 → 進入隨機生成的迷宮 → 清除樓層 / 擊敗Boss → 提升道具能力。隨著系列發展，每一代都在此基礎上增加新機制（路線分岐、議會、逆海盜、自動研究、道具轉生等），使得道具界系統越來越深度且豐富。對於追求極限能力的玩家來說，道具界是不可或缺的核心內容。
