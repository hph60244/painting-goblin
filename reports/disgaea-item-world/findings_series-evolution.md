# Findings: Item World Evolution Across Disgaea Series

## Disgaea 1: Hour of Darkness (2003)
- Original Item World concept introduced
- 30/60/100 floors based on rarity
- Basic floor progression with Item General/King/God bosses every 10 floors
- Geo Panel and Geo Symbol mechanics
- Innocents (specialists) introduced
- Bonus Gauge system
- Mr. Gency's Exit introduced

## Disgaea 2: Cursed Memories (2006)
- Felony system added (related to Item World progression)
- Land of Carnage introduced (harder version of Item World for high-level items)
- Innocent system refined
- Pirate encounters introduced

## Disgaea 3: Absence of Justice (2008)
- Class World introduced (similar to Item World but for leveling classes)
- Land of Carnage returns
- Evil Symbols introduced (e.g., Nether Express for warp points)

## Disgaea 4: A Promise Unforgotten (2011)
- Split paths introduced: Item Growth Path vs Innocent Boost Path
- Item World Senate / Cam-Pain HQ (pass bills to boost stats)
- Level Spheres (Green/Yellow/Red +5/+10/+15)
- Event Areas (purple columns spawning rewards)
- Mystery Rooms expanded (Ambush Room, Fortune Teller)
- Reverse Pirating system introduced
- Pirate Crew creation
- Maximum item level: 300

## Disgaea D2: A Brighter Darkness (2013)
- Normal/Rare items can be upgraded to Legendary by completing all floors
- Item World can change item rarity
- Maximum item level: 300 (999 in Rasetsu mode)
- Refined speed-running strategies (Masked Hero with fists)

## Disgaea 5: Alliance of Vengeance (2015)
- Cheat Shop allows adjusting Item World size (small maps for speed)
- Nether Express evil symbol for warp station creation
- Curse system added to Item World
- Item World Research feature (send squads on automated expeditions)
- Enemy strength modifiers via Cheat Shop

## Disgaea 6: Defiance of Destiny (2021)
- Switch to 3D graphics
- Infinite floors (no longer capped based on rarity)
- Item World Research expanded (passive leveling of equipment)
- Auto-mode/auto-battle supported
- Maximum level raised to 99,999,999 (character level, not item level)
- Simplified innocent/stat system compared to earlier games
- Mr. Gency's Exit still essential
- Healers important since HP doesn't restore between floors

## Disgaea 7: Vows of the Virtueless (2023)
- Item World returns with further refinements
- "Item Reincarnation" mechanic added (items can reincarnate to gain more power)
- Jumbification (超巨大化) mechanic affects Item World
- Further QoL improvements for speed-running

## Key Trends Across Series
1. Each iteration adds more depth and complexity to Item World mechanics
2. Speed-running optimization tools increase (Cheat Shop, Evil Symbols, auto-battle)
3. Later games emphasize automation (Item World Research squads)
4. The core loop remains the same: enter item → clear floors → kill bosses → level item
5. Maximum potential of items increases with each game
6. Disgaea 6 simplified some mechanics while others (D7) added them back
