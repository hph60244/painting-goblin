# 研究计划：Token性價比高的Model給Coding Agent使用（中國區域）

## 主要研究問題
尋找Token性價比高的Model給Coding Agent使用，特別注意中國區域的資訊，用繁體中文報告。

## 子主題
1. **中國地區可用編碼模型概覽**
   - 調查中國地區可訪問的編程相關AI模型（如DeepSeek、Qwen、GLM、ChatGLM、CodeGeeX等）
   - 了解各模型的API可用性、支援平台、官方文檔
   - 預期資訊：模型列表、提供商、基本特點

2. **Token定價比較**
   - 收集各模型API的定價資訊（每百萬token成本）
   - 比較不同提供商（OpenAI、Anthropic、DeepSeek、百度文心、阿里通義、騰訊混元等）
   - 預期資訊：定價表格、免費額度、速率限制

3. **編碼能力性能評估**
   - 查詢各模型在編程任務上的基準測試成績（HumanEval、MBPP等）
   - 分析性能與成本的關係（性價比）
   - 預期資訊：性能數據、性價比排名

4. **中國區域訪問與限制**
   - 調查中國用戶訪問這些API的具體限制（IP限制、支付方式、審查等）
   - 了解替代訪問方案（國內鏡像、代理服務等）
   - 預期資訊：訪問難易度、推薦方案

5. **綜合推薦**
   - 綜合以上資訊，提供針對中國用戶的性價比推薦
   - 考慮不同使用場景（個人開發、企業應用、高頻使用等）
   - 預期資訊：具體推薦模型、使用建議

## 研究方法
- 每個子主題委派一個研究子代理
- 每個子代理使用3-5次網頁搜索
- 子代理將結果保存到獨立文件：`findings_[子主題編號].md`
- 最後綜合所有結果產生繁體中文報告

## 預期成果
- 完整的模型比較表格
- 性價比分析
- 中國地區使用建議
- 繁體中文研究報告
