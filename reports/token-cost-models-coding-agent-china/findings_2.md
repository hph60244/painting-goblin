## 免費額度與速率限制詳情

### OpenAI
- **免費額度**：API 無永久免費額度，新註冊帳戶可能有試用金。
- **速率限制**：依帳戶層級（免費層、付費層、企業層）設定每分鐘請求數（RPM）與每分鐘 tokens 數（TPM）。
- **區域加價**：資料駐留與區域處理端點加收 10%。

### Anthropic
- **免費額度**：Claude.ai 免費版提供有限對話次數，API 無永久免費額度，新用戶可獲試用 tokens。
- **速率限制**：依方案（免費、Pro、Max、企業）設定使用上限。
- **區域加價**：美國專用推理價格為標準價格的 1.1 倍。

### DeepSeek
- **免費額度**：新用戶註冊後可獲得贈送餘額（具體數量依活動而定），用完後需充值。
- **速率限制**：預設為 5 RPM（每分鐘請求數），可透過申請提升限額。
- **區域加價**：無區域加價，全球統一價格（美元計價）。

### 百度文心（千帆）
- **免費額度**：新用戶可獲 100 萬 tokens 免費額度（部分模型共用），有效期一年。
- **速率限制**：依 QPS（每秒查詢數）限制，不同組件有不同免費調用次數（如百度搜索 100 次/天）。
- **區域限制**：主要服務中國大陸用戶，需中國手機號或企業認證。

### 騰訊混元
- **免費額度**：首次開通服務後獲得 100 萬 tokens 免費額度（混元生文與多模態模型共用），有效期一年。
- **速率限制**：預設接口並發數為 5，可購買並發增購包（800 元/並發/月）。
- **區域限制**：僅在中國大陸提供服務，需騰訊雲實名認證。

## 中國區域訪問注意事項

### 支付方式
- **國際提供商**（OpenAI、Anthropic）：需國際信用卡（Visa/Mastercard），可能受中國外匯管制影響。
- **中國提供商**：支援中國大陸銀行卡、支付寶、微信支付，企業用戶可對公轉帳。

### 網路訪問
- **國際 API**：在中國大陸直接訪問可能不穩定，需使用代理或專線。
- **中國 API**：在中國大陸延遲低，但境外訪問可能受限。

### 合規要求
- **中國提供商**：需實名認證（個人/企業），並遵守內容審核政策。
- **國際提供商**：需注意數據出境合規（如《個人信息保護法》）。

## 總結建議

### 針對中國用戶的 Coding Agent 選型
1. **預算有限、主要中文環境**：首選 **Hunyuan‑a13b** 或 **ERNIE 4.5 Turbo**，成本最低且訪問流暢。
2. **需要國際模型、兼顧成本**：選擇 **DeepSeek**，價格接近中國模型且全球可用。
3. **追求最高編程品質**：考慮 **GPT‑5.4 mini** 或 **Tencent HY 2.0 Think**，在性能與價格間取得平衡。
4. **企業級、高穩定性需求**：使用 **百度千帆** 或 **騰訊混元** 企業版，享有 SLA 與專屬支援。

### 免費額度利用策略
- 新用戶應優先使用各平台的免費額度進行測試。
- 百度、騰訊的 100 萬 tokens 免費額度可覆蓋中小型專案初期使用。
- DeepSeek 的贈送餘額適合快速驗證 API 集成。

### 成本控制技巧
- 啟用快取功能（如 DeepSeek 快取命中價格僅 $0.028/百萬 tokens）。
- 選擇合適的模型規格（如上下文長度、輸出長度）以減少 token 消耗。
- 監控使用量，設定預算告警。
