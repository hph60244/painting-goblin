# 《Ratopia》資源生產與加工建築系統研究報告

## 一、建築分類概述

《Ratopia》中的建築物分為以下七大類別：
- **Base（基礎設施）**：倉儲、床鋪、方塊、梯子、電梯等
- **Raw materials（原物料建築）**：採集、狩獵、農場、牧場等
- **Production（生產加工建築）**：鋸木廠、窯爐、熔爐、紡織廠、麵包店等
- **Service（服務建築）**：廁所、浴場、馬戲團、學校、醫院等
- **Military（軍事建築）**：訓練營、軍營、路障、陷阱等
- **Decoration（裝飾建築）**：壁紙、長椅、噴泉、雕像等
- **Royal（皇家建築）**：研究桌、皇家廚房、皇家 forge、監獄、銀行等
- **Electronics（電子建築）**：發電機、電網、工廠、無人機機庫等

## 四、生產鏈與資源轉換關係

### 4.1 木料生產鏈
```
原木/藤蔓 → 鋸木廠 → 木材、木棍
                         ├→ 木工坊 → 工具、木桶、齒輪
                         ├→ 窯 → 木炭（用於熔爐冶煉）
                         └→ 建築材料
```

### 4.2 金屬生產鏈
```
銅礦/鐵礦/金礦 + 木炭 → 熔爐 → 金屬錠
                                  ├→ 鐵匠舖 → 武器、工具
                                  ├→ 鑄幣廠 → 硬幣（資金來源）
                                  └→ 珠寶工坊 → 珠寶
```

### 4.3 紡織生產鏈
```
草/纖維/藤蔓 → 紡織廠 → 布料、繩索、絲綢
                         ├→ 裁縫店 → 衣物
                         └→ 染料工坊 → 染料 → 印刷廠
```

### 4.4 食品生產鏈
```
穀物農場 → 穀物 → 磨坊 → 麵粉 → 麵包店 → 麵包、蛋糕、冰淇淋
                                              （附屬產品：起司）
獵人小屋/牧場 → 生肉 → 屠宰場 → 肉品、骨頭
                                  ├→ 燒烤店 → 熟食
                                  └→ 磨坊 → 骨粉（肥料）
蘑菇農場（消耗木材）→ 蘑菇
蜂房 → 蜂蜜 → 釀酒廠 → 啤酒
```

### 4.5 石材生產鏈
```
岩石 → 石工所 → 石磚（建築材料）
砂岩 → 石工所 → 沙
                      └→ 玻璃工坊 → 瓶子、飾品
```

### 4.6 貿易與經濟循環

遊戲中經濟系統的核心機制：
- 老鼠執行採集、搬運、建造等指令時，國庫支付「Pia」給老鼠
- 加工品送入倉庫時，政府以 Pia 買入
- 老鼠使用物品或材料時，需支付 Pia 給國庫
- 持續生產無人使用的物品將導致國庫空虛

**中期資金策略**：建立鑄幣廠、熔爐、窯的生產線 → 進口木材與銅礦/金礦 → 生產木炭 → 冶煉金屬錠 → 鑄幣，有效增加貨幣供給量。

**貿易推薦商品**：
- 水晶樹/珊瑚樹加工（玻璃工坊、珠寶工坊）
- Joy Flower 加工（磨坊）
- 啤酒與蜂蜜製品

## 五、遊戲特色與關鍵機制

1. **研究系統**：每種資源首次存入倉庫獲得 1 研究點，於研究桌解鎖新建築藍圖
2. **經濟平衡**：必須平衡國庫與老鼠財富，避免過度生產導致破產
3. **水源重要性**：取得水源後農場與牧場才能穩定運作；冬季前須確保供水
4. **物流優化**：建議將生產建築集中於主倉庫附近，長距離運輸需預留電梯空間
5. **工作條件設定**：可設定生產數量條件，避免過度生產浪費材料與人力
6. **皇家廚房**：專屬料理系統，可透過特定配方順序獲得額外增益效果
