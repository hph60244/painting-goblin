## 一、建築分類總覽

Ratopia 的建築共分為 **8 大類別**，透過研究桌消耗研究點數逐項解鎖藍圖：

| 分類 | 說明 | 典型建築 |
|------|------|---------|
| **基礎（Base）** | 基礎設施、床鋪、地板牆壁、運輸裝置 | 軟床、雙層床、木梯、倉庫、監視哨 |
| **原料（Raw Materials）** | 採集、狩獵、種植、畜牧等初級資源 | 採集營、伐木營、獵人小屋、穀物農場、漁業 |
| **生產（Production）** | 加工製造，將原料轉為中間材或成品 | 鋸木廠、紡織廠、熔爐、麵包店、造幣局 |
| **服務（Service）** | 滿足市民歡樂、衛生、教育等需求 | 廁所、澡堂、雜技台、學校、酒吧 |
| **軍事（Military）** | 防禦與戰鬥相關 | 訓練營、軍營、柵欄、陷阱 |
| **裝飾（Decoration）** | 純美觀用途，提升城市美觀度 | 壁紙、畫作、噴泉、旗幟、紀念碑 |
| **王室（Royal）** | 核心管理與領袖專用設施 | 城市入口、研究桌、法典石、稅務局、銀行、貿易中心 |
| **電子（Electronics）** | 電力生產、傳輸、儲存與工業化設施 | 發電機、風力渦輪、太陽能板、電池、無人機機庫 |

另有後期擴充的**咒術（Spell）** 分支，透過咒術桌進行研究。

## 二、建造機制與限制

### 2.1 資源成本
每個建築都需要特定材料，材料分為三個層級：
- **天然資源**：木材、石頭、葉子、泥土
- **加工中間材**：木板、石材、布料、銅錠、鐵板、齒輪
- **高階零件**：電池、馬達、特殊晶片

### 2.2 地形要求
- 建築需要**腳踏實地（足場）或天花板（天井）** 才能放置
- 植物或既有物體佔用的格子需先清除，系統會自動下達挖掘指令
- 遊戲包含**地上與地下雙層地圖**，地下可開採稀有礦物，需梯子或電梯連接

### 2.3 科技解鎖
- 透過**研究桌**消耗**研究點數**解鎖藍圖
- 每**首次**將新物品存入倉庫可獲得 1 點研究點數（共約 150 種物品）
- 研究速度：1 點需 **3 小時**；領袖智力每級提升速度 **+5%**
- 每個研究桌可同時**預約 3 項**研究

### 2.4 建築尺寸
每種建築有固定的**寬×高**（格數）及**基礎設施評分（Infrastructure Score）**，影響城市繁榮度發展。

## 三、工作效率與人力配置

### 3.1 市民三屬性
| 屬性 | 影響 |
|------|------|
| **力量（Strength）** | 負重上限、近戰攻擊力；適用鋸木廠、挖礦等勞力工作 |
| **敏捷（Dexterity）** | 移動速度；適用生產精密物品的建築 |
| **智力（Intelligence）** | 經驗值獲取量；適用服務業與研究型建築 |

屬性每級效果：力量負重 +0~1 / 攻擊力 +1；敏捷移動速度 +2%~3%；智力經驗獲取 +5%。

### 3.2 工作效率公式
- 對應屬性 0 時：每分鐘產出約 **2 單位勞動力**
- 每 1 點對應屬性：每分鐘額外 +0.1 勞動力（+5%）
- 屬性 10 時：每小時勞動量約 **180 單位**

### 3.3 影響效率的因素
- **快樂度（Happiness）**：由體力、食物、歡樂、衛生、日用品五項需求決定。快樂度低則效率顯著下降，極低時可能叛亂。
- **體力系統**：每小時消耗 3 點，睡眠每小時恢復 12 點；體力低於 30 自動就寢，高於 80 醒來。
- **特質（Traits）**：市民隨機攜帶特質，如「靈巧」生產速度 +20%、「笨拙」生產速度 -20%。
- **經驗值與等級**：每工作 5 分鐘獲得 1 點經驗值（受智力補正）。

### 3.4 工作條件設定
玩家可對每個生產建築進行詳細設定：
- 啟用/關閉建築運作
- 設定生產數量上下限（少於 X 個時開始生產）
- 設定材料數量檢查（避免過度生產浪費資金）
- 可將設定**複製**到同類型建築

### 3.5 Ratron 機器人自動化
後期可解鎖 **Ratron 機器人**（最多 30 個），不需工資、食物、睡眠、娛樂，僅需充電站。分為力量型、敏捷型、智力型與戰鬥型，搭配無人機機庫可執行搬運、採礦、維修、建造等任務。

### 4.3 六大生產鏈

**木料鏈：**
```
原木/藤蔓 → 鋸木廠 → 木材、木棍 → 木工坊 → 工具、木桶、齒輪
                               → 窯 → 木炭（熔爐燃料）
```

**金屬鏈：**
```
銅/鐵/金礦 + 木炭 → 熔爐 → 金屬錠 → 鐵匠舖（武器）
                                     → 鑄幣廠（資金）
                                     → 珠寶工坊（珠寶）
```

**紡織鏈：**
```
草/纖維/藤蔓 → 紡織廠 → 布料、繩索、絲綢 → 裁縫店（衣物）
                                          → 染料工坊（染料）
```

**食品鏈：**
```
穀物農場 → 磨坊 → 麵粉 → 麵包店（麵包、蛋糕、冰淇淋）
獵人小屋/牧場 → 屠宰場 → 肉品 → 燒烤店（熟食）
                       → 骨粉（肥料）
蜂房 → 釀酒廠（啤酒）
```

**石材鏈：**
```
岩石 → 石工所 → 石磚（建材）
砂岩 → 沙 → 玻璃工坊（瓶子、飾品）
```

**貿易經濟循環：**
```
原料採集 → 加工製造 → 成品銷售（國庫收購）
                                ↘ 出口貿易獲利
```

## 九、建築佈局策略

### 9.1 核心原則
1. **關聯設施就近擺放**：需水的農場靠近儲水槽；鋸木廠靠近伐木場；熔爐靠近礦區
2. **垂直分層規劃**：地下過度擴張會導致殭屍活躍度上升，建議一半建築建在上層
3. **半徑 40 格原則**：警察局有效範圍內集中擺放行政管理建築

### 9.2 運輸基礎設施
- 礦車軌道提供水平高速移動
- 升降平台負責垂直運輸
- 搬運站可指定物品搬運，搬運工速度 +20%、搬運量 +3

### 9.3 農業配置建議
- 菜園（1×1 / 2×1）：靈活但冬季停止生長
- 農場（4×2）：集中高效但需水源與肥料，物流壓力大
- 使用 Ratbot 機器人可解決農場物流問題

### 9.4 監控系統配置
- 監視哨：左右建築效率 +30%，但被監視者快樂 -5
- 監視攝影機：耗電無需人員，推薦 4 格網格佈局最大化覆蓋

### 9.5 市民分配建議（依階段）

## 十、科技樹與繁榮度解鎖

### 10.1 三大科技樹
1. **基礎科技**：分為基礎、生產、服務、軍事、皇家等類別
2. **工程科技**：透過工程工作檯研究，解鎖電力系統
3. **宗教科技**：解鎖十二生肖神社與兩種宗教系統

### 10.2 關鍵科技路徑建議
- 早期優先：獵人小屋 → 屠宰場 → 兔子牧場 → 紡織廠 → 石匠所
- 法典石碑 → 監獄 → 訓練營 → 稅務署
- 儲水槽 → 穀物農場 → 草地

## 十一、經濟循環與建築關聯

### 11.1 資金流動模型
```
市民工作（採集/搬運/建造） → 國庫支付工資（Pia）
市民從倉庫取用資源 → 向國庫付款
服務建築使用 → 市民支付使用費
生產建築向國庫購買原料 → 成品賣回國庫
                                      ↘ 課稅 / 貿易出口 / 鑄幣維持平衡
```

### 11.2 稅收策略
- 建議稅率：對資產 500 以上徵收 **25%~35%** 財富稅，週期 5 天
- 福利法：對資產 200 以下者每日補助 100~300 Pia
- 中期：建立鑄幣廠生產鏈，進口木材與銅/金礦 → 木炭 → 金屬錠 → 鑄幣

### 11.3 貿易推薦商品
- 水晶樹/珊瑚樹加工品（玻璃工坊、珠寶工坊）
- Joy Flower 加工品（磨坊）
- 啤酒與蜂蜜製品

### 11.4 物流優化
- 將生產建築集中於主倉庫附近
- 長距離運輸需預留電梯空間
- 使用搬運站指定物品運輸路線

## 十二、總結與建議

### 12.1 初期（1-20 人）
- 專注食物安全：採集營 → 獵人小屋 → 穀物農場
- 基礎加工：鋸木廠 → 紡織廠 → 石匠所
- 盡快建造研究桌解鎖更多藍圖

### 12.2 中期（20-50 人）
- 建立鑄幣廠解決資金問題
- 法典石碑 + 稅務局建立稅收制度
- 鐵路與升降機提升物流效率
- 訓練營 + 陷阱應對定期入侵

### 12.3 後期（50+ 人）
- 工程科技樹解鎖電力系統
- Ratron 機器人實現自動化生產
- 特斯拉塔與無人機機庫實現全自動化
- 十二生肖神社提供全方位加成
- 選擇一種紀念碑路線邁向勝利

### 12.4 關鍵提醒
- 不要過度擴張地下，以免殭屍失控
- 持續監控國庫資金，避免過度生產無人使用的物品導致破產
- 快樂度是生產力的基石，務必滿足市民五項基本需求
- 提前規劃垂直運輸空間，後期改造難度極高
