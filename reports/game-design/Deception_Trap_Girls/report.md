# 《影牢 トラップガールズ》（Deception Trap Girls）陷阱調查報告

## 一、遊戲概覽

| 項目 | 內容 |
|------|------|
| 正式名稱 | 影牢 トラップガールズ |
| 開發商 | 光榮特庫摩（コーエーテクモ） |
| 平台 | DMM Games（網頁瀏覽器遊戲） |
| 類型 | 陷阱動作 / 社交網頁遊戲 |
| 服務期間 | 2015年7月30日 ～ 2016年7月29日（約1年） |
| 資料來源 | ピクシブ百科事典、Pixiv 粉絲創作（小說/插畫） |

這是《影牢》系列的社交網頁遊戲衍生作品。與傳統系列最大的不同在於：**陷阱全部擬人化**為女性角色「メディウム（Medium）」，並且具有**衣裝破壞（衣装ブレイク）**系統——當傷害累積到一定程度，Medium 角色的衣服會破損。

## 二、陷阱系統特色

本作繼承了影牢系列的經典陷阱機制，但以抽卡（gacha）收集擬人化陷阱角色為核心。陷阱仍分為三大類別：

1. **地面陷阱（床設置型）**
2. **牆壁陷阱（壁設置型）**
3. **天花板陷阱（天井設置型）**

另有一批**特殊陷阱**，無法簡單歸入上述三類。

## 四、陷阱機制推測

基於影牢系列傳統及粉絲創作內容：

1. **三類別配置**：每場戰鬥可在場地中佈置地面、牆壁、天花板陷阱各若干個
2. **連鎖系統（Combo）**：陷阱可依照順序連鎖觸發，形成連擊
3. **冷卻時間**：每個陷阱使用後需要一定時間重新充能
4. **傷害類型**：
   - **直接傷害型**：メガロック、ニードルフロア、アローシューター 等
   - **位移控制型**：アタックウォール、スプリングフロア、スマッシュフロア 等
   - **束縛控制型**：ベアトラップ、スパイダーネット 等
   - **搞笑型**：タライ、トイハンマー、スノーボール 等（傷害較低但帶有特殊效果）
5. **衣裝破壞系統**：Medium角色受到傷害累積到閾值時，衣裝會依序破損
