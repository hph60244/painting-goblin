# 《悪代官3》陷阱系統分析報告

> 用於同人遊戲開發參考
> 調査日期：2026-06-06

---

## 一、遊戲概述

| 項目 | 內容 |
|------|------|
| 遊戲名稱 | 悪代官3 (Akudaikan 3) |
| 平台 | PlayStation 2 |
| 發售日期 | 2007年3月1日 |
| 開發商 | Global A Entertainment |
| 遊戲類型 | 時代劇喜劇動作遊戲 (前作為時代劇SLG) |
| 導演理念 | "主角是個體弱多病的廢柴" / "陷阱要看起來好玩" / "讓玩家從外部觀看都能發笑的蠢遊戲" |

《悪代官3》是系列第三作，但故事完全重啟，與前兩作無關。系統進行了大幅改革——從2D俯瞰視角改為全3D，從"準備階段佈陷阱→敵人來襲"改為即時佈置、即時戰鬥。

## 二、核心玩法機制

### 2.1 即時陷阱系統（最大改革）

**前兩作：**
- 戰前有5分鐘準備階段，用一萬兩預算佈置陷阱
- 敵人 arriving 後陷阱已固定，不可移動
- 玩家引誘敵人踩陷阱

**本作（3代）：**
- **無準備階段** — 敵人與玩家同時進入地圖
- 陷阱可隨時隨地即時佈置與觸發
- 每個房間最多同時存在3個陷阱
- 每個陷阱有冷卻時間（放置後須等待才能觸發）
- 使用「ワナワナ君」（Ebisuya開發的裝置）來佈置和觸發陷阱

### 2.2 體力病（持病）

- 主角有心臟疾病，連續奔跑會增加心臟負擔（畫面上方臉部圖示顯示）
- 負荷達到最大時，主角會氣喘吁吁，無法移動一段時間
- 慢走或站立可恢復
- 可在選項中關閉（但會影響評價）

### 2.3 屋敷增築系統

遊戲全部發生在惡代官的宅邸內。可使用資金增建房間：
- 草庵、大浴場、拷問部屋
- 西洋風格房間、SF風格實驗室

**房間專用陷阱：** 許多房間有內置的即死陷阱（不需要手動放置，滿足條件自動觸發）：
- 從二樓扔下
- 雪隠流し（掉進糞坑）
- 石抱（跪坐石頭酷刑）
- 木馬責め（木馬酷刑）
- 大浴場淋浴機關

### 2.4 敵人耐性系統（全新）

敵人具有屬性/類型耐性。如果使用被抵抗的陷阱類型攻擊，敵方會：
- 自動閃避
- 瞬移到遠處
- 連擊中斷

因此玩家需要嘗試不同陷阱找出敵人的弱點屬性。

### 2.5 連段計時器擴展

- 前兩作：連段之間有固定時間限制
- 本作：可使用定身陷阱、毒狀態或其他效果來延長連段計時器

### 2.6 護衛系統改革

- 前兩作：準備階段配置護衛，固定站位
- 本作：按房間即時召喚，每房間最多5人，只要有錢可無限召喚
- 被擊敗的護衛會掉落金幣補充當前資金

## 三、陷阱分類與機制

### 3.1 按觸發模式分類（繼承自系列傳統）

| 類型 | 說明 | 範例 |
|------|------|------|
| 踩踏觸發 | 敵人踩到陷阱位置即生效 | 捕獸夾、地雷、落穴 |
| 開關觸發 | 需要手動或自動觸發開關 | 滾石、燭台、大砲 |
| 範圍主動攻擊 | 偵測範圍內敵人自動攻擊 | 吹箭人偶、蜂巢 |

### 3.2 按作用分類

| 類型 | 說明 |
|------|------|
| 傷害型 | 直接減少敵人HP |
| 騷擾型 | 減少精神力（防禦狀態的敵人不會踩陷阱） |
| **屈辱型（3代新增）** | 不傷HP，累積屈辱值使敵人"心折" |

### 3.3 屈辱系陷阱（3代特有）

這是本作最大的創新。通過羞辱敵人使其「心が折れる（心折）」：
- 敵人會撤退、自殺或投降（無論剩餘多少HP）
- 部分心折的敵人會要求成為護衛
- 可招募為護衛，但僱傭費用極高、HP大幅降低

**屈辱武器範例：**
- ピコピコハンマー（敲擊槌）
- 豆鉄砲（豆子槍）
- 罵倒用語集（辱罵詞典 / 挑釁）

### 3.4 元素屬性系統

飛行道具可通過元素孔疊加屬性：
- **火焰** — 點燃敵人，可引爆炸藥桶
- **冰凍** — 凍結敵人，延長連段時間
- **毒素** — 持續傷害，削弱敵人

對應的屬性陷阱：
- 噴火孔 / 噴火童子
- 噴冰孔 / 噴冰童子
- 噴毒孔 / 噴毒童子

## 四、連鎖陷阱設計（核心玩法）

### 4.1 基礎連鎖原理

連鎖陷阱以**捕獸夾**為核心。基本思路：

1. 捕獸夾放在道路中央
2. 周圍四個方向佈置攻擊型陷阱
3. 敵人被夾住 → 發動攻擊 → 形成連鎖

**基礎五連鎖範例（來自系列前作，設計思路可沿用）：**

```
配置：捕獸夾（中央）→ 左側炸藥桶 / 右側燭台（開關在旁） / 再右側滾石（朝向炸藥桶）
流程：惡代官引誘 → 敵人踩夾 → 按X啟動滾石 → 滾石壓點火開關 → 燭台點燃敵人 → 敵人著火引爆炸藥桶 → 滾石碾壓 → 五連擊
```

### 4.2 高級連鎖原則

1. 捕獸夾放在四方向都有空地的位置（留給直線攻擊陷阱）
2. 使用迴轉床讓滾石來回碾壓
3. 利用元素孔疊加屬性傷害
4. 所有開關可由一顆滾石依次觸發
5. 蜂巢和誘導人偶可斜向補刀
6. **注意：** 開關不要放在敵人追擊必經之路上，避免提前觸發

### 4.3 3代連鎖變化

- 3代即時佈置，可在戰鬥中根據敵人位置靈活佈陣
- 利用定身陷阱（捕獸夾、麻痺等）延長連段時間
- 注意敵人屬性耐性，選對陷阱類型

## 六、同人遊戲設計建議

### 可借鑑的設計元素

1. **即時陷阱佈置系統** — 比戰前準備更靈活、更符合現代遊戲節奏
2. **屈辱/心折系統** — 非致命解決方案，增加遊戲策略深度
3. **屬性耐性** — 強迫玩家嘗試不同陷阱組合
4. **房間專用陷阱** — 鼓勵玩家探索和建設基地
5. **多路線劇情** — 提高重玩價值
6. **捕獸夾+連鎖核心設計** — 簡單易懂的連鎖基礎，進階組合多樣

### 需要注意的設計陷阱（避坑）

1. 3代缺乏場景多樣性（全部在宅邸），同人遊戲應注意場景變化
2. 即時佈置可能導致節奏混亂，需要合理冷卻時間控制
3. 屬性耐性系統需要清晰可視化提示
