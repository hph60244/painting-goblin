# Research Plan: Pygame Demo Games for Coding Agent

## Main Research Question
尋找數十個適合讓Coding Agent用Pygame做成demo的小遊戲
(Find dozens of small games suitable for a Coding Agent to make demos with Pygame)

## Subtopic 1: Classic Arcade Games Suitable for Pygame
Investigate classic arcade games (Pong, Space Invaders, Snake, Pac-Man, Tetris, Breakout, etc.) that have simple mechanics and are commonly implemented as Pygame tutorials. Focus on game mechanics complexity and typical implementation scope.

## Subtopic 2: Modern Casual & Mobile Game Concepts
Research modern casual games (Flappy Bird, Doodle Jump, Crossy Road, match-3, endless runners, etc.) that have straightforward mechanics suitable for Pygame implementation. Focus on core game loops and minimum viable feature sets.

## Subtopic 3: Puzzle, Board, & Novelty Game Ideas
Find puzzle games (Sudoku, Minesweeper, Solitaire, Memory card games), simple board games (Checkers, Connect Four, Tic-Tac-Toe), and novelty mini-games that can be implemented in 1-2 days using Pygame. Focus on scope management and clear win/lose conditions.
