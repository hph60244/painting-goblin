# Research Plan: Pygame Game Jam Starter Templates

## Main Question
尋找Pygame的game jam starter，近3年還有在更新，提供原型製作用的功能 (Scene management, Video settings, Audio settings, Input mapping, Save/Load, etc.)

## Subtopics

### Subtopic 1: pygame-ce starter templates & boilerplates
Investigate pygame-ce (community edition) starter templates and boilerplates on GitHub that have been updated in the last 3 years. Focus on templates providing scene management, configurable video/audio settings, and input mapping.

### Subtopic 2: Pygame game frameworks & engines
Research higher-level Python frameworks/engines built on top of pygame that provide rapid prototyping features for game jams: scene management, asset handling, audio, input, save/load systems.

### Subtopic 3: Popular Pygame game jam resources & tutorials
Search for curated lists, articles, and tutorials recommending pygame templates for game jams, including feature comparisons.

## Synthesis
Compare findings across all subtopics, listing top candidates with their features, last update date, and strengths/weaknesses for game jam use.
