# Research Plan: AI Code Agents for Mobile Game Development

## Main Research Question
尋找讓Code Agent根據需求文件製作手機遊戲的參考範例 (Finding reference examples of Code Agents creating mobile games from requirement documents)

## Subtopics

### Subtopic 1: AI Code Agents for Game Development
Investigate existing AI code agents (e.g., Claude, Cursor, GitHub Copilot, Devin, etc.) used to generate mobile games from specifications. Look for tutorials, blog posts, and documentation showing the process of feeding requirements to an AI agent and getting a playable mobile game.

### Subtopic 2: Tools & Frameworks for AI-Generated Mobile Games
Research tools, platforms, and frameworks that enable AI-powered mobile game creation (e.g., Unity ML-Agents, GameBuilder Studio, Luma AI, Scenario, Layer, etc.). Focus on those that take text requirements as input and produce game code/assets.

### Subtopic 3: Case Studies & Examples of AI-Generated Games
Find concrete examples and case studies of mobile games (or simple games) created with AI assistance. Look for GitHub repositories, demo videos, articles documenting the development process from requirements to finished game.

## Synthesis
Compare findings across subtopics to provide actionable guidance on how a code agent can be used to create a mobile game from requirement documents.
