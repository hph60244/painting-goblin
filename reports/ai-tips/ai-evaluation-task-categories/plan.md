# Research Plan: AI Agent / LLM Evaluation Task Categories

## Main Question
What are all the detailed task categories used in evaluating AI agents and LLM models? Compile a comprehensive master table of task types.

## Subtopics

### 1. Major LLM Benchmarks & Their Task Categories
Investigate major LLM benchmarks (MMLU, HumanEval, GSM8K, HellaSwag, ARC, TruthfulQA, BIG-bench, etc.) and extract all task categories they test.
Expected output: List of benchmarks mapped to their specific task categories.

### 2. AI Agent Benchmarks & Their Task Categories
Investigate agent-specific benchmarks (SWE-bench, AgentBench, WebArena, GAIA, ToolBench, API-Bank, etc.) and extract task types.
Expected output: List of agent benchmarks and the task categories they evaluate.

### 3. NLP Core Evaluation Dimensions
Identify the fundamental cognitive/skill dimensions tested: reasoning, comprehension, generation, knowledge retrieval, translation, summarization, etc.
Expected output: A taxonomy of core evaluation dimensions with descriptions.

### 4. Domain-Specific & Specialized Evaluations
Cover specialized domains: coding, math, medical, legal, finance, multilingual, safety/alignment, instruction following, long context, etc.
Expected output: Domain-specific task categories and evaluation approaches.

## Synthesis Plan
Merge all findings into a single comprehensive master table organizing tasks by category, with sub-tasks, example benchmarks, and descriptions.
