# Research Plan: AI Code Agent Web Game Development

## Main Research Question
尋找讓Code Agent根據需求文件製作網頁遊戲的參考範例
(Find reference examples of letting a Code Agent create web games based on requirement documents)

## Subtopic 1: AI Code Agents Generating Web Games from Specs
Investigate known examples and tools (e.g., Claude, Cursor, Copilot, GPT-4, Cline/Coding Agent) being used to generate complete web games (HTML/JS/CSS) from natural language descriptions or requirement documents. Look for case studies, blog posts, videos.

## Subtopic 2: Workflow Patterns for LLM-Based Game Development
Research structured approaches to prompt engineering for game creation — how to write requirement documents/specs that yield good results from code agents. Look for system prompts, spec templates, iterative development patterns.

## Subtopic 3: Best Practices and Frameworks
Identify recommended workflows: iterative prompting, testing strategies, project structure for AI-generated games. Look for guidelines on using AI coding agents effectively for game development projects.
