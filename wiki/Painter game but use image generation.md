- https://boardgamegeek.com/boardgame/290236/canvas

- [[Remake game but use image generation]]

- 模仿桌遊的形式
	- 每次開局形式都相同，使用相同規則
	- 只要達成分數條件後就可以通關
		- 不使用解謎遊戲的硬性條件
- 玩家成長
	- 通關的獎勵可以換技能
	- 技能可幫助玩家更容易通關
- 遊戲設置
	- 開局抽選(3)個任務
	- 發給玩家手牌
- 遊戲進行
	- 玩家使用行動條進行動作
		- 行動條規零時遊戲失敗
	- 每次行動可以
		- 抽(1)張手牌
	- 隨時可以
		- 調整牌堆
		- 完成任務
		- 使用技能
- 牌張
	- 上半部是角色圖，跟主遊戲無關
		- 美觀用途
		- 加分用途
	- 下半部有(5)個色塊放成橫排
		- 遊戲進行時最多挑(5)張牌放入牌堆
		- 牌堆有順序性，因此上面的牌的色塊會擋住下方牌的色塊
		- 只要從最上面看起來色塊與任務相同，即可用來完成目標
		- 色塊共有(5)整顏色，寬高比為1:2
